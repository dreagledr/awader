# AWADER

## Описание

Небольшая программа с ипользованием либы kit

## Структура проекта

- `main.c` - основной файл программы
- `kit.h` - заголовочный файл с основными функциями
- `assets.h` - файл с ресурсами
- `compile.bat` - скрипт для компиляции проекта Tiny C Compiler, но подойдет любой c99 компилятор под windows
- `awader.exe` - исполняемый файл
